VSCodium

I have spent some time looking around the internet to find a modern editor or IDE which would suit my needs for this project. I have always found Intellij from JetBrains a very good IDE for java development but with this project I wanted something a kin to a smart editor rather than an IDE. I looked at KATE and like the simple design and cross platform use, particularly on linux but it does nor lend itself to web development. My other favourite is Note++ but again not suitable my requirements. I then started using VSCode from microsoft, this I found ticked all the boxes but the AI style tools suggesting code completion became very wearisome. It may have been my initial install making VSCode unsuitable and a tweak in settings menu woulld have solved it. I finally opted for VSCodium, which is an open source off-shoot of VSCode. I was careful with the installation to tailor the editor to my requirement. I currently think VSCodium is the best editor out there.

javascript

After years and years of server based program development, I have often looked down on javascript as a development language. With my latest project I have discovered its rightful place on the client side. Javascript seems to be the glue between HTML 5 and CSS to provide a richer user experience.
Javascript

The page you are current looking at provides my first attempt at javascripting. It was a very useful introduction for me. I scouted the internet for a page which had the basic functionality I needed. I then copied the code and went through it line by line, documenting what each line did. Once I understood the code in detail, I modified it and added additional code to provide the page you see now. This page provides information in a single column table. Within each row there is a header label which groups the rows detail by subject. I have provided 2 filters, one allows all row of a common subject to be displayed and then a further sub-filter to select from within the details
Using javascript was a throw back to my early years in programming equating to something like a 2 or 3 Generation Language. I enjoyed the experience, the big plus was the language’s simplicity. The other useful feature which I found was the use of the browser console log for debugging.
